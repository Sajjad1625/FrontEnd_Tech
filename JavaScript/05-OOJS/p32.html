Static Method..!

Js provides static method that belongs to the class instead of an instance of class . These methods are called directily on the class itself.

Points to remember..!

  • The static keyword is used to declare a static method.
  • The static method can be any name.
  • A class can contain more than one static method.
  • If we declare more than one stativ method with a similar name, th JS always invokes the last one.
  • We can use this keyword to call the static method within another static method.
  • we can't use this keyword directily to call a static method within non-static method.
  • In such case, we can call the static method either using the class name or as the property of the constructor
