This is CANVAS TAG

The HTML canvas element provides HTML a bitmaps surfaces towork with. It is used to draw graphics on the web page. The HTML 5 canvas tag is used to draw graphics using scripting language like JavaScript.

This is a first canvas tag

Note: It is always necessary to specify the id attribute and the height & width attribute to define the size of the canvas. You can have multiple canvas elements on one HTML page.
